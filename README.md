# Tech Media AI Bot 🤖📱

Полностью автоматизированный Telegram-бот и AI-контент-менеджер для ведения технологического канала. Бот генерирует стильные, экспертные и полезные посты о технологиях, Android, iPhone и гаджетах, используя нейросеть Google Gemini.

## 🌟 Возможности
- **Генерация постов по рубрикам:** Android, iOS, сравнения, фишки, приложения.
- **Фирменный стиль:** Все посты пишутся в едином стиле современного tech-медиа (заголовок, суть, "почему это важно", итог).
- **Автопостинг:** Бот сам генерирует и публикует контент в канал через заданный интервал времени.
- **Без баз данных:** Максимально легкий проект, состояние хранится в памяти. Идеально для бесплатных серверов.
- **Удобный интерфейс:** Управление через Reply-кнопки прямо в Telegram.

## 📂 Структура проекта
Проект состоит всего из 3 файлов:
- `main.py` — вся логика бота (хэндлеры, генерация, автопостинг).
- `requirements.txt` — список зависимостей.
- `README.md` — эта инструкция.
